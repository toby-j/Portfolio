Python tool

This tool needed to perform a series of checks while navigating to defined areas of Sony's TV software.

We used the producer-consumer model which suited our multiprocessing design of the software. This gave us enough performance to process each frame in a queue without progressively falling behind the real time frames of the TV software.

We are also able to successfully build an object detection algorithm that'll navigate to areas of the software from any start point.
